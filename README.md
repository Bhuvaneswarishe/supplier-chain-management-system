# SCM Delivery Mismatch + Global Chatbot

## Backend

1. Create `.env` from `.env.example`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Start the API:
   `uvicorn app.main:app --reload --port 8000`

## Frontend

1. From `frontend/`, install dependencies:
   `npm install`
2. Start Vite:
   `npm run dev`
